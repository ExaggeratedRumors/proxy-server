# Requirements

- JDK 20
- Gradle 8.4

# Execution

1. Run server
```bash
./gradlew :server:run
```
2. Run clients

```bash
./gradlew :client:run
```

# Modules

- client - client source code
- server - server source code
- commons - common DTO, objects and configuration
